## Installation

> Install with yarn or npm: `yarn` or `npm`:

```bash
# yarn
yarn add five-star
```

```bash
# npm
npm i five-star --save
```

### Import the lib with es6 or cjs:

```mjs
// es6
import StarModule from 'five-star';
```

```cjs
// cjs
const StarModule = require('five-star');
```
